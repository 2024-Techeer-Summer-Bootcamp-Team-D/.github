# ğŸš€ How To Start

### 1. Clone The Repository
```
https://github.com/2024-Techeer-Summer-Bootcamp-Team-D/MVStudio-Docker.git
```
### 2. ENV Setting In The MVStudio-Docker Folder
* .env
```
ELASTIC_VERSION=
ELASTIC_PASSWORD=
LOGSTASH_INTERNAL_PASSWORD=
KIBANA_SYSTEM_PASSWORD=
METRICBEAT_INTERNAL_PASSWORD=
FILEBEAT_INTERNAL_PASSWORD=
HEARTBEAT_INTERNAL_PASSWORD=
MONITORING_INTERNAL_PASSWORD=
BEATS_SYSTEM_PASSWORD=
INIT_INDEX=
```
* MVStudio-Frontend/.env
```
VITE_REACT_APP_BASE_URL =
```
* MVStudio-Backend/.env
```
SECRET_KEY=
DEBUG=True

MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=
MYSQL_USER=
MYSQL_PASSWORD=
DATABASE_HOST=
DB_SQL_MODE=

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_STORAGE_BUCKET_NAME=
AWS_S3_REGION_NAME=

OPENAI_API_KEY=
SUNO_API_KEY=

SOCIAL_AUTH_GOOGLE_OAUTH2_KEY=
SOCIAL_AUTH_GOOGLE_OAUTH2_SECRET=
RUNWAYML_API_KEY=

BASE_BACKEND_URL=
BASE_FRONTEND_URL=
REFRESH_TOKEN_SECRET=
KAKAO_APP_ADMIN_KEY=
CID=
```
<br>

### 3. Run Docker
```
$ docker-compose -f docker-compose.prod.yml -f docker-compose.monitoring.yml up -d --build
```
